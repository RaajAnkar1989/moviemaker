# 🎬 AI Movie Maker Pro

A professional cinematic video generation suite that works as both a **Native macOS Desktop App** and a **Mobile-Friendly Web App**.

## 🌟 Features

- **Industry Transitions**: 10+ pro transitions including Glitch, Whip Pan, Cross Dissolve, and more.
- **Smart Zoom**: Automatically removes watermarks from video edges.
- **Multi-Page Credits**: Create movie-style titles and end credits with multiple pages and lines.
- **Audio Mixing**: Balance background music and original video volume.
- **Mobile Friendly**: Deploy to the web for access on any phone.

## 🚀 How to Run

### 🖥️ On macOS (Desktop App)
1. Ensure you have Python installed.
2. Install dependencies: `pip install -r requirements.txt`
3. Double-click `MovieMaker.command` to launch.

### 📱 On Web/Mobile (Streamlit)
1. Install dependencies: `pip install -r requirements.txt`
2. Run the web server: `streamlit run movie_maker_app.py`

## ☁️ Deployment

To host this for mobile users:
1. Push this folder to a GitHub repository.
2. Connect the repo to [Streamlit Community Cloud](https://share.streamlit.io).
3. Select `movie_maker_app.py` as the main file.
4. Share your public URL!

## 📦 Dependencies
- `moviepy` (v2.0+)
- `customtkinter`
- `streamlit`
- `numpy`
